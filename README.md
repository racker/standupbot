# Installation Notes


## MacOSX

`$ brew install node`

`$ brew install npm`

`$ GOTO Common`

## Linux Ubuntu 12.04

`$ sudo apt-get update`

`$sudo apt-get install g++ curl libssl-dev apache2-utils git-core make`

`$ wget http://nodejs.org/dist/v0.8.14/node-v0.8.14.tar.gz`

`$ tar xzvf node-v0.8.14.tar.gz`

`$ cd node-v0.8.14`

`$ ./configure`

`$ make`

`$ sudo make install`

GOTO Common

## Common
`$ npm install js-yaml fs express irc`
